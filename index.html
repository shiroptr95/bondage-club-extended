(B) Ручной, через панель закладок (Mobile: Android):

1) Скопируйте все из следующего текстового поля:

2) Нажмите 3 точки -> Звездочка закладки -> Изменить (в нижней части экрана во всплывающем окне).

Что бы активировать:
1) Войти в клуб
2) Нажмите на строку URL
3) Напишите название сохраненной закладки
4) Нажмите на него (слева будет звездочка)
*) Не пытайтесь выбрать закладку из списка закладок, это не сработает!

(B) Ручной, через панель закладок (Mobile: iOS):

1) Скопируйте все из следующего текстового поля:

2) Нажмите «Поделиться» -> «Добавить закладку» и сохраните.
3) Откройте список закладок -> нажмите «Изменить» (внизу экрана во всплывающем окне)

1) Войти в клуб
2) Открыть список закладок
3) Нажмите на закладку, которую вы сохранили

Собранные данные

Переведённая версия BCX содержит изменения только в интерфейсе, все API, в т.ч. сбора данных, оставлены оригинальные, по этому собранные данные отправляются разработчику оригинального BCX (Jomshir98).
BCX Собирает некоторые данные для внутренней статистики. Все данные хранятся у первых лиц и используются только для статистики и принятия решений в целях развития.
Данные собираются исключительно с помощью внутриигрового механизма «Beep» BC, что означает, что любые данные за пределами BC (например, IP-адреса) не собираются и не могут быть собраны.
Полный список собираемых данных:
  • Версия BCX и BC
  • При использовании обычной или разрабатываемой версии BCX
  • Какая ссылка использовалась для загрузки BCX (или что она была загружена из другого мода)
  • Заголовок UserAgent вашего браузера
  • Если в общественной комнате (и пространстве комнаты) или в частной комнате - получается автоматически через данные звукового сигнала.
  • Номер участника учетной записи, отправляющей звуковой сигнал, — получается с помощью данных Beep и используется исключительно для предотвращения дублирования записей.
Они собираются как часть механизма «Проверка версий». Вы можете проверить код, отвечающий за это ЗДЕСЬ
